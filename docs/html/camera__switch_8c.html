HAL_StatusTypeDef Check_Camera_State ( struct cameraSwitch  camSw)

Function to check is there more than one channel set to 1 (SET).

Parameters
structCameraSwitch camSw Functions takes CameraSwitch structure as parameter.

As cameraSwitch structure represents switch it cannot have set state on more than one pin. This function was created to check correctness of pins readings. If there are situation that there were more than one pin read as set, it throws error. Function is importatnt to have mechanizm blocking sending message to mainboard with incorrect information - it could make errors in rover driver.
