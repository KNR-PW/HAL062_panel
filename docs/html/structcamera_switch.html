Detailed Description

Struct representing single camera screen with switch to changing image (one screen handles 3 cam).

A,B,C,D channels are simply read pin state connected with 4-position switch. It allows to contain information with camera image is currently shown on screen. cameraNumber - number of screen that struct is connected to
