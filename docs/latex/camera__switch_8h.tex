As \doxylink{structcamera_switch}{camera\+Switch} structure represents switch it cannot have set state on more than one pin. This function was created to check correctness of pins readings. If there are situation that there were more than one pin read as set, it throws error. Function is importatnt to have mechanizm blocking sending message to mainboard with incorrect information -\/ it could make errors in rover driver.
